# SoccerScore

Aplicativo Android simples para simular o placar de uma partida de futebol.

## Funcionalidades

- Tela 1: formulário com nomes dos times e gols (validação de campos obrigatórios e gols >= 0)
- Tela 2: resumo da partida e opção de editar ou confirmar
- Tela 3: resultado final (vitória do time A, vitória do time B ou empate) e opção de iniciar um novo jogo

## Tecnologias

- Kotlin
- Jetpack Compose
- Navigation Compose

## Como executar

1. Abra o projeto no Android Studio
2. Rode no emulador/dispositivo

Opcional via terminal:

```bash
./gradlew installDebug
```
